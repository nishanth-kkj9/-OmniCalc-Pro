# OmniCalc Pro
An all-in-one advanced professional calculator for Windows, built with Python & PyQt5.

## Features
- Basic, Scientific, Graph, Unit Converter, Programmer, Matrix, Statistics, Finance calculators
- History tracking with SQLite
- Dark/Light modern Windows 11 style UI
- Custom frameless window, keyboard shortcuts, clipboard support
- Export graphs, copy results, auto-save settings

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run application: `python main.py`
3. Build executable: `pyinstaller --onefile --windowed --icon=assets/icons/app.ico main.py`
